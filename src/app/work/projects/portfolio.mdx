## Overview

This project was a personal challenge to design, develop, and deploy a complete portfolio website within **just one hour** — using the support of **AI design and code generation tools**. While the content is placeholder (dummy), the site serves as a proof-of-concept for how fast, efficient, and creative modern web workflows can become with the right automation stack.

## Goals

- Showcase my capabilities in AI-driven development
- Build a visually appealing and functional UI
- Create an editable template for future real content
- Test the limits of how much can be shipped in one hour

## AI Tools & Workflow

- **Prompt Engineering**: Used structured prompts to guide generative design and content creation.
- **Layout Generation**: Leveraged AI to generate grid-based sections, animations, and layout logic using HTML/CSS/JS frameworks.
- **Text Filler Automation**: Populated the entire website with intelligent placeholder content for sections like projects, about, contact, and blog.
- **Deployment**: Hosted via Vercel with GitHub integration for version control.

### Stack

- **Next.js & React**: Frontend framework for rapid builds
- **Tailwind CSS**: Utility-first styling for fast UI composition
- **ChatGPT & AI Design Assistants**: For layout suggestions, dummy data, and logic scaffolding
- **Vercel**: For instant deployment and preview URLs

## Challenges

- **Time Constraint**: Strictly 1 hour — required intense focus and lean decisions
- **Precision Under Pressure**: Balancing aesthetics and structure in a minimal time window
- **Design Feedback Loop**: Iterative fine-tuning with AI suggestions — some prompts worked, others had to be adapted on the fly

## Outcome

The result is a sleek, AI-assisted portfolio that showcases what’s possible in minimal time with the right toolchain. While the content is temporary, the structure is production-ready and adaptable for future personal branding.

- **Total Time**: 60 minutes  
- **Pages Built**: 4 (Home, About, Projects, Contact)  
- **User Feedback**: Positive responses on design flow, color use, and responsiveness

## What I Learned

- AI is a force multiplier — not a replacement. It accelerates execution but still requires human taste and curation.
- Planning is everything when speed is the constraint.
- Dummy content can still tell a compelling story if structured meaningfully.
