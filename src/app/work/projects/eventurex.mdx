## Overview

The **Eventure X** is a modern agile project management application inspired by Jira, built using Next.js 14 with App Router architecture and Appwrite as the backend platform.

The system enables teams to manage workspaces, projects, tasks, and team collaboration through a scalable cloud-native architecture. It demonstrates how modern full-stack tooling can replicate enterprise-grade workflows while maintaining developer-friendly structure and performance optimization.

---

## Core Objectives

- Recreate core Jira-like project management workflows
- Implement scalable workspace-based architecture
- Integrate authentication, real-time data handling, and collaboration tools
- Build a full-stack application using modern serverless backend infrastructure
- Ensure responsive and accessible user interface design

---

## Core Features

### Workspace Management

- Multi-workspace support
- Invite-based collaboration
- Role-based membership system

### Project Management

- Project creation with visual identity
- Task organization across multiple projects
- Structured workspace hierarchy

### Task Management

- Status workflow:
  - BACKLOG
  - TODO
  - IN_PROGRESS
  - IN_REVIEW
  - DONE
- Drag-and-drop task management
- Due date tracking
- Assignee management

### Authentication System

- OAuth login support (Google & GitHub)
- Secure session management
- Appwrite authentication workflows

### Analytics & Dashboard

- Task analytics overview
- Workspace switching
- Responsive dashboard views

---

## Architecture & System Design

The application follows Next.js App Router architecture:

- Modular feature-based folder structure
- Dedicated auth and dashboard routes
- Server and client component separation
- API route handling via dynamic routing

### Backend (Appwrite)

- Cloud database collections:
  - Tasks
  - Projects
  - Members
  - Workspaces
- Storage bucket for image uploads
- OAuth integrations
- Permission-based data access
